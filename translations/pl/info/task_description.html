Państwa funkcja powinna przyjąć ciąg znaków jako dane wejściowe i przekonwertować wszystkie pierwsze litery słów w ciągu znaków na wielkie litery, czyniąc ciąg znaków tytułem (pozostałe litery muszą być małe).

Dane wejściowe: Oryginalny ciąg znaków (str{% if is_js %}ing{% endif %}).

Wyjście: Przekonwertowany ciąg znaków (str{% if is_js %}ing{% endif %}).

Przykłady: 

{{init_code_tmpl}}

Jak jest używany:

  • do przetwarzania tekstu i zadań normalizacji danych;
  • do renderowania tekstu w interfejsie użytkownika w standardowym formacie wielkości liter.

Warunki wstępne:

  • sentence ∈ string;
  • length(sentence) >= 0.
